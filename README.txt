== Description ==

Sara Ajax Search is a live search plugin for wordpress themes.It performs real time search as you enter anything.it supports group search like post,page and custom post type in your wordpress website.it's a fully responsive.

==feature==
* Live search
* Post,Page and Custom Post Type search supported
* Dynamic Limit display results.
* Hide and show thumbnails
* Hide and show post Title
* automatically Hide and show  Content After Each WordPress Post or Page
